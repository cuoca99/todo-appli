# TODO アプリ

React + Vite で構築されたシンプルなTODOアプリケーションです。

## 機能

- **TODO追加**: 新しいTODO項目を追加
- **TODO表示**: すべてのTODO項目を一覧表示
- **TODO編集**: 既存のTODO項目のテキストを編集
- **TODO削除**: 不要なTODO項目を削除
- **完了/未完了の切り替え**: チェックボックスでTODOの完了状態を管理
- **データ永続化**: ローカルストレージを使用してデータを保存
- **統計情報**: 全体、完了、未完了の件数を表示

## 開発環境のセットアップ

依存関係は既にインストール済みです。

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてアプリにアクセスできます。

## ビルド

```bash
npm run build
```

ビルドされたファイルは `dist` ディレクトリに出力されます。

## プレビュー

ビルド後のアプリをプレビューする場合:

```bash
npm run preview
```

## 技術スタック

- **React 18**: UIライブラリ
- **Vite**: ビルドツール
- **CSS**: スタイリング（モダンなダークテーマ対応）

## 使い方

1. 上部の入力フォームに新しいTODOを入力し、「追加」ボタンをクリック
2. チェックボックスをクリックしてTODOを完了/未完了に切り替え
3. 「編集」ボタンでTODOのテキストを編集（完了済みのTODOは編集不可）
4. 「削除」ボタンでTODOを削除
5. 下部の統計情報で進捗を確認

データはブラウザのローカルストレージに自動保存されるため、ページを更新しても保持されます。
